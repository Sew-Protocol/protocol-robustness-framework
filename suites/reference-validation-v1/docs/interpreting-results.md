# Interpreting Results

## Status values

- **PASS**: scenario/claim/invariant satisfied under pinned assumptions.
- **FAIL**: mismatch against expected deterministic evidence artifacts.
- **INCONCLUSIVE**: reserved for future suites with partial evidence states.

## Classifications

- **regression-protected**: high-value behavior locked by expected artifacts.
- **bounded-counterfactual**: adverse strategy represented and bounded.
- **capacity-bounded**: liveness/capacity assumptions hold in modeled limits.
- **safety-critical / fund-safety / escalation-robust**: trust-critical safety categories.

Reserved/future terms: ROBUST, BRITTLE, COUNTEREXAMPLE.
