# Bundle Verification Spec

A bundle root (generated by `run-and-report`) can be verified without the Clojure runtime using only standard JSON + SHA-256.

## Algorithm

```
1. Load bundle-root.json
2. Strip keys "bundle/id" and "bundle/hash" from the top-level map
3. Serialize the remaining map as canonical JSON:
   - Sorted keys at all nesting levels (alphabetical)
   - No whitespace between tokens
   - Keywords (in Clojure) become strings
   - Strings escaped per JSON spec (\ -> \\, " -> \")
4. Compute: SHA-256("BUNDLE_ROOT_V1" || canonical-json-utf8-bytes)
5. Compare the 64-char lowercase hex digest against "bundle/hash"
   from the original bundle
```

## Python 3 (zero external dependencies)

```python
import json, hashlib

with open('bundle-root.json') as f:
    bundle = json.load(f)

preimage = {k: v for k, v in bundle.items()
            if k not in ('bundle/id', 'bundle/hash')}
canonical = json.dumps(preimage, sort_keys=True,
                       separators=(',', ':'), ensure_ascii=False)
digest = hashlib.sha256(b'BUNDLE_ROOT_V1' + canonical.encode()).hexdigest()

expected = bundle.get('bundle/hash', '')
if digest == expected:
    print('PASS: bundle hash verifies')
else:
    print(f'FAIL: computed {digest}, expected {expected}')
    exit(1)
```

## Shell (jq + openssl)

```bash
jq 'del(."bundle/id", ."bundle/hash") | tojson | 
    openssl dgst -sha256 -binary | 
    xxd -p' bundle-root.json
```

## Clojure (requires PRF)

```clojure
(require '[resolver-sim.run.bundle-root :as br])
(br/compute-json-hash bundle-root)
```
